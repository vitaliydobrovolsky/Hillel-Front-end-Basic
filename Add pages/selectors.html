Type of selectors CSS

Next will be the list of slectors types:

  • * - all elements are selected for changes
  • TAG (h1) - select elements by tag name
  • class (.class) - select elements by class name
  • atribute (target="") - select elements with atribute name
  • id (#...- select element by ID
  • mix of tags - select all selected tags(a, h1, h2...)
  • path: #selectors>ul>li - indicate path to required elemnt
